Escaneo de Puertos en Python

El escaneo de puertos es una técnica utilizada en ciberseguridad para detectar qué servicios están abiertos en un host. Este script básico en Python escanea puertos comunes para evaluar la seguridad de una red.

Este ejemplo en Python utiliza la biblioteca `socket` para conectarse a diferentes puertos de un host y verificar cuáles están abiertos. Puedes personalizar la lista de puertos o escanear todos los puertos desde el 1 al 65535.

Código de Ejemplo en Python

Código para Escaneo de Puertos:


import socket
from datetime import datetime

# Definir la IP objetivo
target = input("Introduce la IP a escanear: ")

# Puertos comunes a escanear
common_ports = [21, 22, 23, 25, 53, 80, 110, 135, 139, 143, 443, 445, 993, 995, 1433, 3306, 3389, 8080]

print(f"\nIniciando escaneo en {target}...")
start_time = datetime.now()

# Escaneo de cada puerto en la lista
for port in common_ports:
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    socket.setdefaulttimeout(1)
    result = sock.connect_ex((target, port))
    if result == 0:
        print(f"Puerto {port}: Abierto")
    else:
        print(f"Puerto {port}: Cerrado")
    sock.close()

# Tiempo total de escaneo
total_time = datetime.now() - start_time
print(f"Escaneo completado en: {total_time}")

Este código realiza el escaneo de puertos de una IP especificada, mostrando cuáles están abiertos y calculando el tiempo total de ejecución.

Conclusión

El escaneo de puertos es fundamental para pruebas de penetración, ayudando a identificar servicios potencialmente vulnerables. Este script ofrece una base para construir un escaneo de puertos más avanzado.
