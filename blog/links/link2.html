Wikipedia:
SQL Injection - Wikipedia - Artículo que ofrece información sobre la historia, el funcionamiento, ejemplos y métodos de mitigación.

Acunetix:

SQL Injection Attacks by Example:

PortSwigger:

Acunetix Blog:
SQL Injection: What It Is and How to Prevent It - Un artículo que proporciona información sobre la inyección SQL y las mejores prácticas para su prevención.
